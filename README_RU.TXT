Эта программа работает с файлами "version.txt".
Начиная с v1.1 также возможно исправление "Обновление не требуется" на 5.50 GEN.
(Полностью выглядит так:
"На консоли PSP™ установлено
системное программное обеспечение
версии 9.90.
Обновление не требуется.")

Программа поставляется в двух редакциях:
1. OFW_MODE - без операций с flash0, потенциально безопасная
2. CFW_MODE - имеет возможность записи во flash0, не для нубов =)

В общем, теперь можно лёгко и быстро оперировать с файлами!
К примеру, ms0:/seplugins/version.txt для кастома ME...
...или ef0:/seplugins/usbversion.txt для кастома от команды PRO =)
Также, есть и другие пути для старых ХЕНов, к примеру.
Также можно защититься от обновления, записав version.txt версии 9.99 во flash0.

Куда кидать файлы:
[Корень карты_памяти или памяти_устройства]
|-PSP
 |-GAME
  |-V_SWITCH
   |-EBOOT.PBP
   |-version1.txt
   |-version2.txt
   |-version3.txt
   |-version4.txt
